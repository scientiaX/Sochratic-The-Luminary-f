# Waitlist Landing Page

Project ini berisi ekstraksi landing page tanpa dependensi Lottie. Fokusnya adalah layout dan konten utama menggunakan Tailwind CSS.

## Menjalankan dengan Docker

```bash
docker compose up --build
```

Akses di `http://localhost:5173`.

## Menjalankan secara lokal (tanpa Docker)

```bash
npm install
npm run dev
```

## Catatan

Landing page ini tidak menyertakan animasi Lottie. Semua komponen berasal dari halaman landing asli dan disederhanakan untuk pemakaian ulang.
